Início Rápido
1. Configuração

Edite o arquivo frontend/offline.html e altere a URL da API:

const API_BASE_URL = 'http://seu-dominio/api';
2. Acessar

Abra no navegador:

http://seu-dominio/frontend/offline.html
3. Usar
  1. Selecione um evento
  2. Busque um participante
  3. Registre presença
  4. Sincronize quando online
4. Testar

Acesse o painel de testes:

http://seu-dominio/frontend/offline-test.html
Funcionalidades Principais
✅ Carregamento de Dados

Carrega usuários, eventos e inscrições

✅ Registro Offline

Registra presença sem internet

✅ Fila de Sincronização

Gerencia itens pendentes

✅ Sincronização Automática

Envia dados ao servidor quando online

✅ Detecção de Conexão

Verifica status a cada 5 segundos

✅ Interface Responsiva

Funciona em desktop, tablet e celular

Troubleshooting
❓ Dados não carregam?
  • Verifique a URL da API
  • Verifique token em localStorage
  • Abra o console (F12)
❓ Sincronização falha?
  • Verifique conexão de internet
  • Verifique token ainda é válido
  • Veja os logs de erro
